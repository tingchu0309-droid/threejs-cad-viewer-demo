# threejs-cad-viewer

My first Three.js CAD viewer demo built with React and Vite.

## Features

- Three.js scene setup
- Perspective camera
- Grid helper
- Axes helper
- Box geometry demo
- OrbitControls interaction
- Basic engineering-style viewer layout

## Tech Stack

- React
- Vite
- Three.js

## Run locally

```bash
npm install
npm run dev
